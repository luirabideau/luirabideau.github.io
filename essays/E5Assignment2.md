## What did you learn from this assignment?

I learned a lot about passing information between pages. Additionally I learned a lot about routes, and how the server handles different types of requests, and how the server can respond to requests.

## How did you get help when you needed it? What did you need help with?

I asked for help specifically about exchanging information from page to page. This was especially difficult without the use of cookies, but it was doable. 

## How was developing this assignment different than assignment #1?

I think in theory, you would assume this assignment would be easy, however, without the use of cookies, it’s very difficult. Before we started the cookie lab, I really struggled to understand why my code isn't working. However, after the cookie lab everything started to make more sense. 

## Estimate the % of time you spent thinking about how to do something:

Most of my time (75%) was spent brainstorming ideas on how to easily pass data from page to page. 

## Estimate the % of time you spent writing code (but do not include testing:

Only around 5% of my time was actually spent writing code. I used ChatGPT for tedious things but there wasn’t much additional code to add. 

## Estimate the % of time you spent testing and debugging:

20% of my time was spent testing and trying to figure out why something didn’t work. Sometimes it was because I didn’t spell something right, maybe I got a variable wrong. But the majority of the time, I simply didn’t understand how to send information from one place to the other.

## Describe what worked well with this project? What did not work well?

Honestly, because of the limit of working WITHOUT cookies and sessions, it would be best to do the bare minimum. It simply isn’t worth the time. It would make more sense to do okay on this assignment and then spend the time doing all the IRs and EC for Assignment3.
Experimenting is the whole way to finish this project. Nothing can be just inserted in. Everything must be tested and understood. 

## If you could go back in time and do things differently, what would you do differently?

I would’ve started sooner and worked on finding more creative ways to transmit the information from place to place. A method I discovered of handing information from page to page is using hidden input tags in the HTML form. The input takes the value from the query string which is sent to the server through a post message. The server takes the input, and puts it back into the query string and sends the user to the next page. Yes, it’s not the most efficient or best way to do it. But it works. Unfortunately, I discovered that too late.
